# UI 设计

## 页面结构

- 首页：展示系统定位、数据规模、FAISS 状态、Demo 检索和登录入口。
- 登录页：支持登录、注册和访客浏览，公共注册默认创建普通用户。
- 分析工作台：左侧为 UMAP 视图控制，中间为 UMAP 画布，右侧为检索、条件过滤、评测和结果面板。
- AI 辅助分析：复用检索结果，展示问题输入、LLM 状态、UMAP、Top-K 组成、距离摘要和 Markdown 报告。
- 数据集页：展示数据集列表、状态、细胞数、向量维度、样例 Cell ID、元信息维护和生命周期操作。
- 索引页：展示当前索引、索引参数、构建配置、加载已有索引、可用索引列表和删除操作。
- 系统管理页：管理员查看用户、修改角色、启用/禁用或删除用户，并查看系统状态、查询日志和评测日志。

## 交互设计

- 权限不可用的操作使用禁用按钮和说明提示。
- UMAP 支持点选细胞，点选后自动回填查询 Cell ID，并关闭向量输入模式。
- 检索结果会在 UMAP 中高亮查询细胞和 Top-K 命中点，并绘制虚线连接。
- 可视化控制区集中管理着色字段、抽样点数、抽样方式、元数据过滤和基因表达叠加。
- 检索面板支持普通 ANN、向量检索、Exact、ANN vs Exact 和批量评测。
- 数据集和索引页采用“列表 + 详情/操作区”结构，便于演示生命周期管理。

## 视觉风格

整体采用科研分析工作台风格。主色为青绿色和墨色，背景使用浅灰和白色卡片，强调数据密度和可读性。状态使用统一徽标区分 `ready`、`loaded`、`validated`、`offline`、`error` 等状态。

## 可访问性与可用性

- 表单输入提供明确标签。
- 关键操作有加载状态和禁用状态。
- 删除数据集和索引前使用确认框。
- 大图支持缩放、恢复和导出图片。
- 查询结果表格和紧凑预览同时存在，兼顾快速浏览和完整查看。

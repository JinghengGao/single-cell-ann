# 用户手册

## 启动

后端：

```powershell
conda activate single-cell-ann
python -m backend.app
```

前端：

```powershell
cd frontend
npm install
npm run dev
```

浏览器打开 `http://127.0.0.1:5173` 或 Vite 输出的实际地址。

## 账号

- 首次空用户库可创建管理员账号。
- 后续公开注册默认创建普通用户，不能直接注册管理员。
- 管理员可在“系统管理”页修改角色、启用/禁用账号或删除用户。

## 数据集操作

1. 进入“数据集”页。
2. 点击“扫描本地”或上传 `.h5ad` 文件。
3. 勾选数据集并点击“校验选中”。
4. 查看细胞数、向量维度、元数据字段和样例 Cell ID。
5. 维护名称、来源、物种、组织和描述。
6. 点击“加载数据集”或“激活”进入当前工作区。
7. 可将数据集下线、恢复或从登记表删除。

## 索引操作

1. 进入“索引”页。
2. 选择构建模式：联合索引或独立索引。
3. 选择索引类型：IVF_FLAT 或 HNSW。
4. 选择距离度量：L2、Cosine 或 IP。
5. 设置索引参数。
6. 点击“构建索引”。
7. 已保存索引可通过索引 ID 加载，例如 `liver_ivf_flat`。
8. 可在索引列表中切换或删除索引。

如果 IVF 构建失败或进程崩溃，优先加载已有索引或改用 HNSW。

## 检索操作

1. 进入“分析工作台”。
2. 输入 Cell ID，或点击 UMAP 点自动回填。
3. 设置 Top-K。
4. 可选填写 `cell_type`、`disease`、`AgeGroup`、`tissue` 条件。
5. 点击“执行检索”。
6. 查看查询耗时、结果数量、Top-K 表格和 UMAP 高亮。

高级模式：

- “查询向量输入”：直接粘贴与索引维度一致的向量。
- “精确检索”：用暴力 L2 作为基线。
- “ANN vs 精确 对比评测”：查看 Recall 和延迟差异。
- “批量查询评测”：输入多个 Cell ID，查看 QPS、Avg、P50、P99。

## 可视化操作

- 选择着色字段：数据集、细胞类型、疾病、年龄组、组织。
- 设置抽样点数和抽样方式。
- 使用元数据过滤控制可见点。
- 输入基因名或 Ensembl ID，例如 `ALB`，应用基因表达着色。
- 点击 UMAP 点回填查询 Cell ID。
- 点击导出 CSV 保存当前点和元数据。

## AI 辅助分析

1. 先执行一次 Top-K 检索。
2. 进入“AI 辅助分析”页。
3. 输入自然语言问题。
4. 可选择是否启用思考模式。
5. 点击生成报告。

如果未配置 LLM API Key，页面会提示配置缺失；主检索功能不依赖 LLM。

## 系统管理

管理员可执行：

- 查看用户列表。
- 修改角色。
- 启用或禁用账号。
- 删除用户。
- 查看 FAISS、数据集和索引状态。
- 查看最近查询日志和评测日志。

## 常见问题

- Network Error：确认后端正在运行，前端 API 地址指向 `http://127.0.0.1:5000/api`。
- 无法查询：确认已登录、索引 ready、查询 Cell ID 存在。
- 索引无法构建：确认数据集已校验和加载；联合索引要求向量维度一致。
- UMAP 点击无回填：刷新前端页面；当前版本已修复字段不一致问题。
- AI 分析失败：检查 `.env` 中的 `SCANN_LLM_PROVIDER`、`SCANN_LLM_API_URL`、`SCANN_LLM_API_KEY` 和 `SCANN_LLM_MODEL`。

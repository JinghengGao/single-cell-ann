# 项目概述

## 项目目标

本项目实现一个面向单细胞高维向量数据的 ANN 相似细胞检索系统。系统以 `.h5ad` 文件为数据源，读取 `obsm/X_pca` 作为检索向量，读取 `obsm/X_umap` 作为二维可视化坐标，支持数据集管理、索引管理、条件 Top-K 检索、实验评估、UMAP 可视化和 AI 辅助分析。

## 结项要求覆盖

- 系统功能完整：提供可运行的 Flask 后端、React 前端、登录权限、数据集、索引、检索、可视化和管理模块。
- 条件检索：支持按输入细胞或输入向量检索，并可限定 `cell_type`、`disease`、`AgeGroup`、`tissue`。
- 实验评估：返回查询耗时，支持 ANN vs Exact Recall/延迟/speedup，支持批量 QPS、Avg、P50、P99。
- 可视化展示：提供 UMAP 散点图、Top-K 高亮、连线、过滤、基因表达叠加和点击点回填查询。
- 数据集管理：支持扫描、上传、校验、加载、激活、下线、恢复、删除和元信息维护。
- 动态索引管理：支持构建、保存、加载、切换和删除 IVF_FLAT/HNSW 索引。

## 加分项覆盖

- ANN 算法改进：提供 IVF_FLAT 和 HNSW 两类索引，支持 L2/Cosine/IP 度量和参数调优，并通过对比评估展示精度/时间差异。
- 多数据集联合检索：支持多个数据集联合构建索引，实现跨数据集搜索；要求向量维度一致。
- RAG + 单细胞数据库：自然语言问题先抽取 Cell ID 或元数据条件，系统执行 ANN 检索，再将检索证据交给 LLM 生成回答。
- 其他功能：权限管理、管理员监控、查询日志、评测日志、向量检索、批量检索、CSV 导出。

## 技术栈

- 后端：Flask、h5py、NumPy、FAISS、pytest。
- 前端：React、Vite、Axios、ECharts、Lucide React。
- 文件存储：`data/registry.json`、`runtime/users.json`、`indexes/`、`logs/`。

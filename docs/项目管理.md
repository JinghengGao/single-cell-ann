## 验收清单

- 可运行系统：后端、前端、默认数据和页面流程可启动。
- 条件检索：Cell ID + Top-K + 元数据过滤可返回结果。
- 实验评估：单次耗时、ANN vs Exact、批量 QPS/P50/P99 可展示。
- 可视化：UMAP、Top-K 高亮、点击回填、基因表达叠加可演示。
- 数据集管理：增加、校验、加载、编辑、下线、恢复、删除可演示。
- 动态索引：构建、加载、切换、删除可演示。
- 加分项：HNSW/参数调优、多数据集联合索引、AI 辅助分析、管理员监控。

## 风险与预案

- FAISS IVF 崩溃：使用已有索引、HNSW 或 CPU FAISS 环境；演示前不要临时依赖 IVF 现场训练。
- 多数据集演示不足：默认仓库只有 `liver.h5ad`，如要展示跨数据集检索，需要准备第二个向量维度一致的数据集。
- LLM API 不可用：AI 是增强功能，主检索链路不依赖 LLM；可展示错误提示和配置说明。
- 前后端端口变化：确认后端 CORS 包含 Vite 当前端口，必要时重启前后端。
- 大文件加载慢：提前扫描、校验和加载数据，提前准备索引文件。

## 演示建议

优先演示稳定链路：登录、加载已有索引、点击 UMAP 回填、条件 Top-K、ANN vs Exact、批量评测、管理员日志。若现场环境稳定，再演示构建新索引和 AI 分析。

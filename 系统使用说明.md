# 单细胞 ANN 检索系统使用说明

本文说明当前系统有哪些功能、如何导入数据集、如何构建索引、如何查询相似细胞，以及 `cell_id` 从哪里获取。

## 1. 当前功能

系统页面明确分为五个模块：

- 用户信息模块：本地演示账号的注册、登录、退出和当前用户显示。
- 数据管理模块：扫描本地 `.h5ad`、上传 `.h5ad`、校验数据集、选择一个或多个数据集。
- 索引构建模块：使用 FAISS IVF_FLAT 构建 ANN 索引，支持联合索引和独立索引。
- 查询检索模块：输入 `cell_id` 和 Top-K，返回相似细胞及元信息。
- 可视化展示模块：展示 UMAP 抽样点，高亮查询细胞和 Top-K 命中细胞。

核心数据字段：

- 检索向量：`obsm["X_pca"]`
- 可视化坐标：`obsm["X_umap"]`
- 细胞 ID：`obs["_index"]`
- 元信息：`cell_type`、`disease`、`AgeGroup`、`tissue`

## 2. 启动系统

后端：

```powershell
conda activate single-cell-ann
python -m backend.app
```

前端：

```powershell
cd frontend
npm install
npm run dev
```

浏览器打开 Vite 输出的地址，通常是：

```text
http://127.0.0.1:5173
```

## 3. 数据集导入

系统支持两种导入方式。

本地目录扫描：

```text
data/liver.h5ad
data/datasets/*.h5ad
```

把 `.h5ad` 文件放入上述目录后，在页面点击“扫描本地”。默认演示数据 `data/liver.h5ad` 会被登记为 `dataset_id = liver`。

网页上传：

在“数据管理模块”选择 `.h5ad` 文件并点击“上传”。文件会保存到：

```text
data/uploads/
```

上传文件不会提交 Git。

## 4. 数据集校验

选择数据集后点击“校验选中”。系统会检查：

- 是否存在 `obsm/X_pca`
- 是否存在 `obsm/X_umap`
- 是否存在 `obs/_index`
- PCA 向量行数、UMAP 行数、cell_id 数量是否一致

校验成功后页面会显示细胞数、向量维度、元信息字段和示例 `cell_id`。默认 `liver.h5ad` 应显示：

```text
cell_count = 69032
vector_dim = 30
```

## 5. 索引构建

在“索引构建模块”中选择数据集和构建模式。

联合索引：

- 选择一个或多个数据集。
- 系统把这些数据集的 PCA 向量合并成一个 FAISS 索引。
- 查询结果会返回每个命中细胞所属的数据集。
- 多数据集联合索引要求 PCA 维度一致。

独立索引：

- 选择一个或多个数据集。
- 系统为每个数据集分别构建一个 FAISS 索引。
- 可以在索引下拉框中切换当前索引。

默认参数：

```text
index_type = IVF_FLAT
metric = l2
nlist = 256
nprobe = 16
```

## 6. 查询相似细胞

在“查询检索模块”中：

1. 选择查询数据集，或使用自动匹配。
2. 输入 `cell_id`。
3. 设置 Top-K。
4. 点击“查询”。

返回结果包括：

- 排名
- 所属数据集
- `cell_id`
- 距离
- 相似度
- `cell_type`
- `disease`
- `AgeGroup`
- `tissue`
- UMAP 坐标
- 查询耗时

如果多个数据集里存在相同 `cell_id`，需要指定查询数据集，避免歧义。

## 7. cell_id 示例

`cell_id` 来自 `.h5ad` 文件的：

```text
obs["_index"]
```

默认 `liver.h5ad` 可直接测试这些 ID：

```text
AAACCTGAGCAGGTCA-1_2
AAACCTGAGGCCATAG-1_2
AAACCTGCAAGCTGAG-1_2
AAACCTGGTCCAGTTA-1_2
AAACCTGTCATAAAGG-1_2
AAACCTGTCTGGAGCC-1_2
AAACGGGAGCCAGGAT-1_2
AAACGGGAGCGAAGGG-1_2
AAACGGGAGTCCGGTC-1_2
AAACGGGAGTTACCCA-1_2
```

也可以通过命令查看更多：

```powershell
conda activate single-cell-ann
python -c "import h5py; f=h5py.File('data/liver.h5ad','r'); print([x.decode() for x in f['obs']['_index'][:50]])"
```

## 8. API 示例

扫描数据集：

```powershell
curl -X POST http://127.0.0.1:5000/api/datasets/scan
```

校验数据集：

```powershell
curl -X POST http://127.0.0.1:5000/api/datasets/validate `
  -H "Content-Type: application/json" `
  -d "{\"dataset_ids\":[\"liver\"]}"
```

构建联合索引：

```powershell
curl -X POST http://127.0.0.1:5000/api/index/build `
  -H "Content-Type: application/json" `
  -d "{\"dataset_ids\":[\"liver\"],\"mode\":\"combined\",\"nlist\":256,\"nprobe\":16}"
```

构建独立索引：

```powershell
curl -X POST http://127.0.0.1:5000/api/index/build `
  -H "Content-Type: application/json" `
  -d "{\"dataset_ids\":[\"liver\"],\"mode\":\"separate\"}"
```

查询 Top-K：

```powershell
curl -X POST http://127.0.0.1:5000/api/search `
  -H "Content-Type: application/json" `
  -d "{\"dataset_id\":\"liver\",\"cell_id\":\"AAACCTGAGCAGGTCA-1_2\",\"top_k\":5}"
```

获取 UMAP 点：

```powershell
curl "http://127.0.0.1:5000/api/visualization/cells?limit=5000&dataset_ids=liver"
```

## 10. 常见问题

Network Error：

- 确认后端正在运行。
- 确认前端请求的是 `http://127.0.0.1:5000/api`。
- 如果 Vite 自动切换端口，刷新页面或重启后端。

构建索引失败：

- 先扫描并校验数据集。
- 联合索引的数据集 PCA 维度必须一致。
- 检查 FAISS 是否可用：`python -c "import faiss; print(faiss.get_num_gpus())"`。

查询无结果或报 unknown cell：

- 确认 `cell_id` 来自当前选择的数据集。
- 如果使用独立索引，确认当前索引对应的数据集。
